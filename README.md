# Sorting Hat

## Description
This app allows the user to enter a student name into the Sorting Hat. Houses are assigned at random. Students can be sorted by house or expelled.

## How to run
* Clone down this project
* Install [http-server](https://www.npmjs.com/package/http-server) from npm
* At the root of thsi project run the following command: `hs -p 8888`
* In your browser navigate to `https://localhost:8888`
